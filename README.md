■サービス概要
どんなサービスなのかを３行で説明してください

・「AquqZooBridge（仮）」は自分の推しの動物園・水族館を投稿できるサービスです。
・ユーザーは投稿から今度ここにいこうと思えるようになります。
・投稿する側は推しの水族館・動物園を応援できます

■ このサービスへの思い・作りたい理由
このサービスの題材となるものに関してのエピソードがあれば詳しく教えてください。
このサービスを思いつくにあたって元となる思いがあれば詳しく教えてください。

・私は水族館と動物園が好きですが人気な動物よりもマニアックな生き物の方が好きで、どんな生き物を人が好きなのか知りたくなりました。
また、動物園・水族館は見る角度によっても見え方が変わってきます。人が写真を撮っているのを見てどんな写真を撮っているのか知りたいと思いました。
・このアプリを通して動物園・水族館の来館者が増えて、自然や生き物に興味を持つ人が少しでも増えればいいなという願いもあります。

■ ユーザー層について
決めたユーザー層についてどうしてその層を対象にしたのかそれぞれ理由を教えてください。

- 投稿を通じて好きな動物園・水族館を応援できるのでこの層をユーザーとしました。
・週末どこにいくか迷っている人。
- このサービスを通して動物園・水族館に行きたいと思うのでこの層をユーザーとしました。

■サービスの利用イメージ
ユーザーがこのサービスをどのように利用できて、それによってどんな価値を得られるかを簡単に説明してください。

・ユーザーは自分の好きな動物園・水族館を投稿できます。
好きな動物園・水族館を投稿することで来館者が増えて間接的に応援ができます。それによって満足感が得られます。
また、投稿は動的OGPでXシェアすることが可能です。Xを通しての自己開示、交流のきっかけになればと思います。 

・ユーザーは動物園・水族館に関する投稿を見て、今度の週末どこにいくかの選択肢になります。
行ったことある場所でも他の人の視点を見ることで新しい発見につながると考えています。

【ログインしていない場合】  
・投稿一覧、詳細の閲覧  
・タグ絞り込み機能を利用した閲覧  
  
【ログインした場合】  
・投稿一覧、詳細の閲覧  
・絞り込み機能を利用した閲覧  
・動物園・水族館の投稿（項目については検討中） 
　-投稿者のニックネーム  
　-動物園・水族館のタイトル  
　-動物園・水族館の魅力  
　-動物園・水族館の公式url 
　-動物園・水族館の画像のアップロード機能  
　-タグつけ  
・投稿削除、編集  
・投稿いいね  
・投稿のXシェア 
・ラインで友人や恋人に投稿をシェア
・投稿のbookmark
